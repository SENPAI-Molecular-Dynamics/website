MDS file format (Molecular Dynamics Substrate)

This manual covers the MDS file format used by SENPAI to load substrates.
At the end of the manual, the user is able to create their own MDS files to load substrates in SENPAI.

Scope

The MDS file format is used to store information regarding a substrate.

File extension

The file extension for an MDS file is .mds
Files following the MDS standard must have their name terminate with .mds

File content

An MDS file contains:

  1. The substrate name
  2. The file author
  3. A description of the substrate
  4. An "atom" block
  5. A "bond" block
  6. A marker specifying the end of the file

File structure

MDS files are parsed line by line.

Line 1

  • Contains the substrate name
  • Formatted as [string]
  • The string is the whole line

Line 2

  • Contains the file author's identity
  • Formatted as [string]
  • The string is the whole line

Line 3

  • Contains a description of the substrate
  • Formatted as [string]
  • The string is the whole line
