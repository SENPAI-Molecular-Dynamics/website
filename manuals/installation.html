SENPAI installation manual

This manual covers the installation process for the SENPAI molecular dynamics simulator on POSIX systems.
At the end of the manual, the user is left with a useable simulator on the target system.

Required software

  • A POSIX compliant system
  • A C99 compliant compiler
  • A C99 compliant libc
  • GNU Make

Getting the source code

The source code for the latest release can be downloaded from the official senpai website at the following address:

https://www.chelsea486mhz.fr/senpai/source/latest.tar.gz

The wget command can be used to download the archive directly. The tar command is used to unpack the archive:

$ wget https://www.chelsea486mhz.fr/senpai/source/latest.tar.gz
$ tar -xvzf latest.tar.gz
$ cd senpai

Configuring the source code

Edit the file headers/config.h to configure the source code to your liking.
Detailed explanations for each paramater are provided within that file.

$ vim headers/config.h

Building and installing the binaries

With the source code configured, use GNU make to build and install the software:

$ make
$ make install

SENPAI is now installed on the system and ready for use.
